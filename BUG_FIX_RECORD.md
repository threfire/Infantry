# Bug 修复记录

## 1. Pitch 轴斜坡目标跟踪呈阶梯型并滞后

**修复时间**：2026-05-17 04:58:47 +08:00

**问题现象**

pitch 轴目标位置为连续斜坡时，实际位置没有连续跟随，而是呈阶梯型运动并带有明显滞后。VOFA 曲线中，总输出力矩呈波浪型；去掉线型前馈后，PID 输出仍然表现为波浪型。惯量前馈在斜坡匀速段贡献很小，运动主要由反馈输出推动。

**根因定位**

pitch 轴位置反馈使用 MIT 电机编码器相对角，速度反馈使用 IMU gyro Y。位置闭环和速度反馈来自不同测量链路，经过不同传感器、安装位置和机械弹性路径，导致相位和噪声特性不一致。控制器计算 `speed_error = ref_vel - gyro` 时，这个不一致会直接进入 D 项输出，形成周期性的 PID 力矩波动。

原前馈只包含惯量加速度项：

```c
ff_torque = J * ref_accel;
```

目标位置为匀速斜坡时，参考加速度 `ref_accel` 只在起停阶段明显，匀速段接近 0，所以前馈在主要运动阶段无法提供持续力矩。

**修复方式**

pitch 轴速度反馈改为由同一个位置反馈源生成：使用 `relative_angle` 做差分，并经过低通滤波得到 `relative_speed`。pitch 速度误差改为：

```c
speed_error = ref_vel - relative_speed;
```

pitch 前馈从单一惯量前馈扩展为速度前馈加惯量前馈：

```c
ff_torque = PITCH_VELOCITY_FF_GAIN * ref_vel + J * ref_accel;
```

这样斜坡匀速段由 `PITCH_VELOCITY_FF_GAIN * ref_vel` 提供持续前馈力矩，起停阶段由 `J * ref_accel` 补偿加速度需求。

**涉及文件**

- `User/APP/gimbal_task.c`
- `User/APP/gimbal_task.h`
- `User/APP/yaw_pitch_direct.c`
- `User/APP_Support/common/robot_param.h`

**验证结果**

VOFA 观察结果显示，pitch 实际位置跟踪恢复连续，PID 输出波浪明显收敛，前馈在斜坡匀速段持续参与输出。

## 2. Pitch 轴上下行静摩擦补偿不对称导致单向抖动

**修复时间**：2026-05-25

**问题现象**

pitch 轴目标轨迹平滑时，抬升方向跟踪正常，下降方向实际角度出现一抖一抖的波动。

**根因定位**

pitch 轴抬升和下降方向的等效负载不同，统一静摩擦补偿系数会在其中一个方向形成补偿过量或补偿不足。误差靠近死区边界时，补偿量反复进入和退出，叠加 D 项速度反馈后形成单向抖动。

**修复方式**

将静摩擦补偿从 yaw/pitch 共用参数拆分为各轴独立参数，并进一步把 pitch 轴补偿拆成抬升和下降两套系数：

```c
GIMBAL_PITCH_STATIC_FRICTION_COMP_UP
GIMBAL_PITCH_STATIC_FRICTION_COMP_DOWN
```

pitch 补偿计算按误差方向选择对应系数，yaw 轴继续使用独立的 `GIMBAL_YAW_STATIC_FRICTION_COMP`。

**涉及文件**

- `User/APP/gimbal_task.c`

**验证结果**

pitch 轴下降方向抖动来源确认是上下行静摩擦补偿不对称，拆分补偿系数后可单独调整下降方向补偿量。

## 3. 摩擦轮设定线速度与实际弹速存在能量传递效率

**修复时间**：2026-05-25

**问题现象**

摩擦轮目标线速度按 11 m/s 设定后，实际射出弹丸根据弹道反推约为 8.5 m/s，弹速低于设定目标。

**根因定位**

摩擦轮线速度通过接触挤压传递给弹丸，传递过程存在打滑、接触时间限制、压紧量差异和发射管摩擦损耗，所以弹丸出口速度会低于摩擦轮轮面线速度。

当前能量传递效率按速度比估算：

```text
η = 实测弹速 / 设定轮面线速度 = 8.5 / 11 = 77.3%
```

**修复方式**

将目标弹速换算为摩擦轮目标线速度时引入当前传递效率：

```text
目标轮面线速度 = 目标弹速 / η = 11 / 0.773 = 14.24 m/s
目标摩擦轮转速 = 14.24 * 60 / (2π * 0.03) = 4530 rpm
```

**涉及文件**

- `User/APP_Support/shoot_task.h`

**验证结果**

当前传递效率约为 77.3%，若目标弹速为 11 m/s，摩擦轮目标转速应从 3820 rpm 调整到约 4530 rpm 后再实测校准。
